# Code of Conduct

## In Short

Be respectful. Don't be a jerk.

## Details

**We want this to be a welcoming community.**

That means:
- Respect different perspectives
- Accept constructive criticism
- Focus on what's best for the project
- Show empathy toward others

**We won't tolerate:**
- Harassment of any kind
- Trolling or insulting comments
- Personal attacks
- Unwelcome sexual attention

**If someone's being a jerk:**

Contact me directly: Twitter DM @san1302 or email (in GitHub profile)

I'll handle it privately and fairly.

**Scope**

This applies to:
- GitHub issues and PRs
- Project discussions
- Representing the project elsewhere

**Enforcement**

Violations may result in:
- Warning
- Temporary ban
- Permanent ban

Depends on severity and intent.

## Attribution

Inspired by the Contributor Covenant, but simpler.

Our goal: Build a useful tool. Be cool to each other while doing it.
